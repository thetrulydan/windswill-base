# windswill-base Demo App

Interactive demo and testing app for the windswill-base component library.

## Running the Demo

```bash
# From the windswill-base root
pnpm install
pnpm dev --filter demo
```

The demo app will be available at http://localhost:5174

## Purpose

- **Development tool** — Test every component with all variants and states during windswill-base development
- **Starting point** — See how components work and copy patterns for your own project

## Features

- Sidebar navigation with all component sections
- Live theme switcher (light/dark)
- Interactive demos for every component
- Toast notifications for all actions
- Responsive layout

## Sections

- **Foundations** — Colors, Typography, Spacing, Icons
- **Inputs & Controls** — Button, Input, Select, etc.
- **Layout & Navigation** — AppShell, Tabs, Card, etc.
- **Overlays** — Modal, Drawer, Tooltip, etc.
- **Feedback** — Toast, Alert, Progress, etc.
- **Data Display** — Table, Charts, TreeView, etc.
- **Patterns** — Forms, Auth Screen, Empty States, etc.
- **App Modes** — Offline simulation, Theme customization
